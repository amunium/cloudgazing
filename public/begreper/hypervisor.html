Hypervisor

Hypervisor er software som brukes til å distribuere hardware til virtuelle maskiner. Det vil si at hvis man laster ned en hypervisor, så kan dette software fordele hardware som lagringsplass, cpu, ram osv. som maskinen har tilgjengelig, og opprette virtuelle maskiner som kan ta i bruk denne hardwaren. Vi har to typer hypervisors:

Type 2

Type 2 hypervisors er software som blir installert på lik linje som hvilket som helst annet program, og kjører akkurat likt som andre ordinære program. Dette vil si at når hypervisoren er installert og programmet åpnes og skal begynne å fordele hardware, så er mye av hardwaren alt i bruk av datamaskinen fra før.

Type 1

Derfor har vi type 1 hypervisors. De er og software, men lastes ikke ned på lik linje som andre vanlige programmer på en pc. En type 1 hypervisor skreller datamaskinen og tar alt hardware og oppretter virtuelle maskiner direkte oppå hardware.
